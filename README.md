## Запуск

- Скачайте код
- Установите зависимости командой `pip install -r requirements.txt`
- Создайте базу данных и отмигрируйте её командой `python3 manage.py migrate`
- Запустите сервер командой `python3 manage.py runserver`

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, необходимо создать файл `.env` рядом с `manage.py` и записать туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.

Доступны 3 переменные:
- `DEBUG` — дебаг-режим. Поставьте True, чтобы увидеть отладочную информацию в случае ошибки.
- `SECRET_KEY` — секретный ключ проекта
- `DATABASE` — однострочный адрес к базе данных, например: `sqlite:///db.sqlite3`.
